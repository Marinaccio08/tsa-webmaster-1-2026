Framework Statement

This website was built using Node.js, Express.js, and Nunjucks as frameworks, along with Vite for frontend bundling, Firebase for authentication and database services, and Tailwind CSS for styling. All templates, themes, and page designs were created entirely by the team from scratch. No pre-built website templates or pre-designed themes were used at any point in the development of this site.
