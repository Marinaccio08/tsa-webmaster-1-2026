1. Project Overview

This project is a Community Resource Hub web application built for the 2025–2026 TSA Webmaster 1 event. The site serves Monmouth County, New Jersey residents by consolidating local non-profits, support services, community events, programs, and hotlines into a single, searchable platform. Key features include: an interactive resource directory with search and tag-based filtering, a spotlight/highlight section featuring three or more community organizations, a public resource-submission form, a community discussion forum, Firebase-backed user authentication, and customizable user profiles.

The website is built with Node.js / Express on the backend, Nunjucks for HTML templating, Vite for frontend bundling, Firebase for authentication and real-time database, and Tailwind CSS for responsive styling. All code was written from scratch by the team; no pre-built templates or themes were used.
